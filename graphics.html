Graphic Cards

CUDA

Image box filter

Application of filters to an image through the use of shared memory of the CUDA architecture, being able to process several pixels at the same time.

Tonal mapping

Tonal mapping is a process that involves applying local contrast changes to highlight zonally the textures of a photograph. In this practice it has been used to achieve a complete record of all the brightness levels of the original scene in HDR (High Dynamic Range) photographs.

Graphic shaders

Making use of the vertex, geometric and fragment shader, a sphere is tessellated in smaller triangles, which are processed, extruded, and colored and the borders of each of them are defined.
